# 🌿 Leaf Disease Detection

## 📌 Description
This project uses Convolutional Neural Networks (CNN) to classify plant leaf diseases from images.

## ⚙️ Technologies
- Python
- TensorFlow / Keras
- OpenCV

## 📊 Model
- CNN architecture
- Image preprocessing (rescaling, resizing)

## 📈 Results
- Accuracy: ~90% (tuỳ bạn sửa lại số thật)

## 🎯 Application
Helps detect plant diseases for agricultural support.

## 🚀 How to run
1. Open the notebook in Google Colab
2. Run all cells
3. Upload an image to test prediction
